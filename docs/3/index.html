右クリック禁止の愚かさ〜引用ツールの提案〜

　WEBサイトでたまに右クリックを禁止するサイトがあります。その目的は転載させないことでしょう。理解できるのですが、手段が愚かだと思います。転載を禁止することはいいとしても、そのために右クリックを禁止したところでHTMLを閲覧すればコピーできますから。

　そもそも、そんなにコピーされたくないなら公開すべきではありません。公開することの益を狙って公開している以上、公開することによる損害についても覚悟すべきです。インターネットは公共の場であり、他人も存在します。なのに自分の利益を守りたいがために他人の自由を奪おうというのですか？　なんと傲慢で自分勝手な考えでしょう。

　撃っていいのは撃たれる覚悟のあるヤツだけです。自分に都合よくだれかの行為を禁止する。あなたがそう望むなら、あなたが身勝手に禁止することを禁止したがる人が現れても不思議ではありません。いたちごっこになります。禁止では誰も救えません。禁止することは馬鹿げていると思いませんか？

　え、そんな説教は聞きたくない？

　自分が生産したのだから生産者のほうが消費者より偉い。だから生産者のほうに権利があって当然だ。そもそも自分のコンテンツを自分の思うように扱ってなにが悪い？

　たしかに一理ありますね。禁止したいなら禁止すればいいと思います。それでも私は禁止するよりもっといい方法があると思っています。

　いずれにせよ転載されたくないですよね？　自分が苦労して作ったコンテンツをパクられたら腹が立ちますよね？　それなら代わりに引用を提案してはいかがでしょうか。

　引用をワンタッチでさせる機能を追加すれば転載が減らせると思います。

　具体的には次のようなHTMLを作ります。引用したいテキストをマウスで範囲選択すると次のHTMLが自動生成されてクリップボードにコピーされるしくみです。それを読者が自分のサイトにペーストすることで引用させるのです。


<blockquote cite="{{引用元URL}}">
<p>{{引用する文章}}</p>
引用元：<cite><a href="{{引用元URL}}">{{引用元ページタイトル}}</a></cite>
</blockquote>

　ブラウザで表示すると以下のような結果になります。

{{引用する文章}}

引用元：{{引用元ページタイトル}}

　じつはすでにこのページで実装済みです。どこでもいいのでテキストをマウスでドラッグ＆ドロップし、範囲選択してみてください。するとクリップボードに上記のHTMLコードがコピーされています。それを確かめるために、テキストエディタなどでペーストしてみてください。あとはそれを自サイトのHTMLコードへ貼り付ければ引用できます。簡単でしょう？

　またはツイートボタンやいいねボタンなどを設置してもいいですね。ボタンを押して、各自のアカウントで拡散してもらいます。

　このように、簡単に引用できる術を用意してあげれば利用者はそちらを使いたくなるでしょう。引用されたら、引用してくれたサイトからの流入も見込めます。引用機能の実装は、右クリック禁止よりもずっと建設的でWinWinになれる素晴らしい解決方法だと思いませんか？

　そもそも私たちは「いいね！」と思ったら拡散したがりますよね？　「ねぇねぇ、こんなイイの見つけたんだけど！　見て見て！」ってなりますよね？　それを商売に利用すべく口コミやツイッターの拡散を戦略としていたりしますよね？　それは禁止すべきことでしょうか。むしろ推奨されることではないのでしょうか？

　禁止したところで、余計にどうにかして誰かに伝えたがるのではないでしょうか。それが人情であるならば、人の動機や欲望を認めた上で対策するのが合理的ではないでしょうか。いっそ人のためになるしくみを作ってしまうほうが健全で生産的で合理的で持続可能な現実的手段ではないでしょうか。

効果がないこともあるでしょう。それでも…… 　効果がないこともあるでしょう。たとえば広告収入がほしい人で、さらにコンテンツ作成に労力をかけず他人のコンテンツを盗むことで広告収入だけを得ようとする場合です。そのような目的のとき、この方法は通じないでしょう。そのときは法律に則って対処するしかありません。

　ですがそれは右クリックを禁止したところで同じです。HTMLファイルごとコピーされるだけです。最初から盗むつもりの人には何をしてもムダです。公開することをやめる以外に手立てはありません。ならばいっそ彼らのことについては法律にまかせればよいのです。私たちは私たちにできる対処をしましょう。それが右クリック禁止なり引用機能の実装です。そして私は相手がなにを目的にしていようとも、自分がよいと思ったことをすることをおすすめします。自分に誇りをもち、気持ちよく生きるために。

　禁止することは愚行です。禁止ではなく代わりの行為を用意し、そちらに誘導するのが賢く建設的な方法です。

　なぜ人のものをパクるような嫌なやつのために、こちらがそこまで用意してあげなければならないのかって？　それが自分のためになるからです。彼らは私たちが作ったコンテンツに価値があると認めているからパクったのです。嬉しいではありませんか。しかし、まるごと盗作されることは許せません。なので引用してもらおうというのです。

　他人のためは自分のためです。著者のほうから引用機能を用意してあげましょう。すると引用する側は簡単に紹介できるため紹介してくれる人が増えるでしょう。すると著者側は紹介してもらいやすくなり読者が増えます。紹介をみた読者はその内容を知れて喜ぶでしょう。自分よし、相手よし、世間よし。三方良し。だれもが幸せになれるのです。

　自分にできる努力をして改善してみませんか？　それが嫌いだったはずの相手さえもありがたい存在にします。新たな読者を呼び込んでくれるファンになります。安易に禁止してだれかを嫌いパクられて損害を被るより、苦労して対案を作り上げて提示し、皆を味方にしたほうがよいと思いませんか？
